# GDP-Data-extraction
This project focuses on extracting, processing, and analyzing global GDP data using Python. The goal is to identify and visualize the top 10 largest economies in the world, ranked by GDP, as part of a data analysis and learning exercise

In this project, I worked with GDP data of different countries and identified the top 10 largest economies in the world, ranked by GDP value.
The project also includes a graph to visualize the results.

What This Project Does

Loads GDP data using Python

Cleans and processes the data

Sorts countries by GDP (highest to lowest)

Selects the top 10 countries

Displays the result using a bar graph

Tools & Libraries Used

Python

Pandas

Matplotlib
